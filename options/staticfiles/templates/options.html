Header

If the header is sticky it will be fixed on the top of your page content even if you scroll down.

If this is checked, actions' hotkeys will be displayed under the actions's button in the header.

If this is checked, views' hotkeys will be displayed under the views's button in the header.

By default the Flowcus's scripts are automatically added to each page you load, it allows the friction-less magic of this extension. But sometimes you might want to disable this behavior.
