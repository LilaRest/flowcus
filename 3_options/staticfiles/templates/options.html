Behaviors > Actions > Markdown ➜ Obsidian

The name of your local Obsidian vault.

The name of the exisiting folder inside your Obsidian vault in which you want to put the clipped contents to. Must end by /.

An header can be automatically added to a clipped note before it is sent to Obsidian. Using the Nunjucks/Jinja syntax you can customize this header. Just format the header and put data tags.

Here are some available tags :
- {{ now }} to get the current datetime
- {{ title }} to get the content's title
- {{ url }} to get the content's url
- {{ domain }} to get the content's domain name
- {{ publication_datetime }} to get the content's publication datetime
- {{ author }} to get the content's author name
- {{ excerpt }} to get the content's excerpt name
- {{ word_count }} to get the content's number of words
